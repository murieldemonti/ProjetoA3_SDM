# :wine_glass: Wine & Rose :rose:

Sistema de gerenciamento de estoque de adega desenvolvido em Java com interface gráfica em Swing.


## :pushpin: Descrição do Projeto

Este projeto apresenta o desenvolvimento do sistema Wine & Rose, uma aplicação desktop criada para facilitar o gerenciamento de estoque de uma adega de vinhos, oferecendo uma interface intuitiva, controle automatizado e integração com banco de dados MySQL.

O sistema permite realizar buscas por informações específicas, cadastrar, consultar, editar e remover produtos, além de atualizar automaticamente o estoque após vendas e enviar notificações de confirmação por e-mail, contribuindo para maior organização e eficiência no processo administrativo.

A gestão eficiente de estoques é fundamental para o bom funcionamento de estabelecimentos comerciais. Em muitos pequenos negócios, o controle manual ainda é uma prática comum, gerando inconsistências, retrabalho e impactando diretamente a qualidade do atendimento. Diante disso, este sistema busca trazer mais confiabilidade, agilidade e organização às rotinas internas.

O sistema foi desenvolvido seguindo uma arquitetura baseada em Model, View e DAO:

- Model: contém os atributos e métodos das entidades manipuladas no sistema.

- View: interface gráfica construída em Swing, responsável pela interação com o usuário.

- DAO: camada responsável pela comunicação com o banco de dados MySQL, garantindo persistência e integridade dos dados.


## :gear: Tecnologias Utilizadas 
- Java 21

- Swing

- MySQL

- Jakarta Mail

- JDBC

- Arquitetura MVC + DAO


## :star2: Funcionalidades Principais

✔ Cadastro completo de produtos com múltiplos atributos

✔ Atualização automática de estoque após venda

✔ Envio de e-mails automáticos para confirmação

✔ Interface gráfica amigável e minimalista

✔ Validações de campos obrigatórios

✔ Persistência de dados integrada ao MySQL

✔ Feedback visual para o usuário

✔ Regras de negócio para controle de estoque e consistência

O sistema foi pensado para gerentes e funcionários de adegas ou lojas de vinho, oferecendo uma ferramenta simples, eficiente e acessível, mesmo para usuários sem experiência técnica.
